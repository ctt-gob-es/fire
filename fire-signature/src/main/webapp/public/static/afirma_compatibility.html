Requisitos de entorno para la firma con certificados locales

Para la firma con certificado local es necesario cumplir los siguientes requisitos:

  • Navegador web:

    • Microsoft Internet Explorer 8 o superior
    • Microsoft Edge
    • Mozilla Firefox
    • Google Chrome
    • Apple Safari

  • Cliente de firma:

    • AutoFirma 1.6.5 o superior
    • Cliente afirma Android 1.7.0 o superior
    • Cliente afirma iOS 1.7.0 o superior
